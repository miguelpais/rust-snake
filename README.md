# rust-snake
A snake game built in Rust running inside the terminal window.

![alt text](https://github.com/miguelpais/rust-snake/blob/main/screen.png?raw=true|width=300)

## How to run

```terminal
$ cargo build
$ cargo run
```

Play with the arrows keys.
